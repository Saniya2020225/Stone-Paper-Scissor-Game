✊✋✌️ Stone Paper Scissors Game

A simple and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, with the winner determined instantly based on the classic game rules.

🚀 Features

- Interactive and responsive user interface
- Play against the computer
- Random computer move generation
- Instant result display (Win, Lose, or Draw)
- Score tracking
- Smooth and user-friendly gameplay

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript 

📂 Project Structure

Stone-Paper-Scissors/
│── index.html
│── style.css
│── script.js
│── README.md

🎮 How to Play

1. Open the game in your browser.
2. Choose Stone, Paper, or Scissors.
3. The computer will randomly select its move.
4. The game will compare both choices and display the result.
5. Continue playing to improve your score.
